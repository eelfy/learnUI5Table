Access SAP Web IDE

Steps to access SAP Web IDE
Note

SAP Web IDE is no longer available via SAP Business Technology Platform trial accounts. Any references to SAP Web IDE in this documentation are only relevant for you if you have access to SAP Web IDE through a productive SAP BTP account. Please consider SAP Business Application Studio as an alternative. See App Development Using SAP Business Application Studio.

Log on to SAP Business Technology Platform, go to your subaccount, choose Services in the navigation bar of the SAP Business Technology Platform cockpit, and open the detailed information on your SAP Web IDE by choosing the SAP Web IDE Full-Stack tile.

Selecting Go to Service leads you to your personal SAP Web IDE.

Tip

You can bookmark this link to access SAP Web IDE later.
