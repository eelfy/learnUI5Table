Adapting the UI

You can extend and customize your SAP Fiori applications.

For information about the features that you can adapt, see Adapting the UI: List Report and Object Page.

Additional Features in SAP Fiori Elements for OData V2

You can extend and customize SAP Fiori applications using the SAPUI5 Visual Editor in SAP Web IDE.

Note

SAP Web IDE is no longer available via SAP Business Technology Platform trial accounts. Any references to SAP Web IDE in this documentation are only relevant for you if you have access to SAP Web IDE through a productive SAP BTP account. Please consider SAP Business Application Studio as an alternative. See App Development Using SAP Business Application Studio.

For information about the features that you can adapt, see:

  • Creating a Binding Change

  • Adapting the UI: Analytical List Page

For more information about SAP Web IDE, see the documentation for SAP Web IDE on the SAP Help Portal at https://help.sap.com/viewer/p/SAP_Web_IDE.

Additional Features in SAP Fiori Elements for OData V4

Key User Adaptation

Key users can change the UI from the initial state of the app to a personalized view for end users. In the User Actions Menu, key users can then choose Adapt UI and change the user interface of SAP Fiori apps directly. For more information about what you have to consider when developing apps that support key user adaptation, refer to SAPUI5 Flexibility: Enable Your App for UI Adaptation. For more information about key user adaptation, refer to SAPUI5 Flexibility: Adapting UIs Made Easy.
